Burrow — Changelog

Historie verzí v češtině. Co se změnilo a proč.
v2.3.3 aktuální červen 2026
Nové funkce
  • Předání úkolu do terminálového tabu (bez zabití session) — tlačítko Hand off v Mission Control teď adoptuje tu samou živou PTY do reálného terminálového tabu místo aby ji zabilo a znovu spustilo claude --resume. create_pty je idempotentní (živá session → znovupřipojení + přehrání ring bufferu), takže agent pokračuje v tabu bez nového procesu. Mission Control si nechává svoje status listenery (pty-hook-{id} se vysílá všem posluchačům), takže puntík zůstává živý.
  • Zámek vstupu + odznak „in tab" — předaný úkol dostane příznak handedOff (persistován do nového sloupce mission_tasks.handed_off), který zamkne vstupní cesty v Mission Control — pole pro pokračování i připojení vestavěného terminálu — aby do jedné PTY nikdy nezapisovali dva pisatelé. UI ukazuje odznak „in tab" + tlačítko Focus tab pro skok zpět. Příznak přežije restart aplikace.
v2.3.2 červen 2026
Nové funkce
  • Stav „permission" + zvonkový ikonka — Claude Code posílá hook PermissionRequest vždy, když potřebuje souhlas uživatele (spustit příkaz, smazat soubor…). Nově se zobrazuje jako jantarový pulzující bod + ikona zvonku v postranním panelu — vizuálně odlišný od modrého bodu „čeká na další zprávu". Dřív se obojí promíchalo dohromady a nešlo poznat, co agent potřebuje.
  • Čítač zpráv ↺N — každé odeslání zprávy agentovi navýší čítač. V postranním panelu se u tabu zobrazí ↺2, ↺3… pokud jste poslali víc než jednu zprávu a tab je aktivní. Tooltip ukazuje „N messages sent to agent this session". Na první pohled vidíte, kolikátá konverzační výměna právě probíhá.
  • OSC 133 — okamžité detekování příkazů v shellu — shell (zsh, bash, fish, iTerm2) posílá markery C (příkaz začal) a D (příkaz skončil) přes OSC 133. Burrow je teď čte a okamžitě přepíná stav tabu na „busy/idle" — bez čekání na 2s polling. Funguje jen pro terminálové taby bez agenta; agentské taby řídí stav přes hooky.
  • Inline přejmenování tabu v postranním panelu — dvojklik na název tabu ho otevře jako editovatelné pole přímo na místě. Enter nebo blur potvrdí, Esc zruší.
  • Titulek okna OS = aktivní tab — záhlaví macOS okna teď zobrazuje „popis úkolu — workspace", takže je vidět co agent dělá bez přepnutí do Burrow.
  • OSC 7 — živé sledování adresáře (CWD) — shell emituje OSC 7 po každém cd. Burrow ho čte a aktualizuje leaf.cwd ihned, bez pollingu. burrow spawn --cwd tak vždy dostane správnou cestu i když agent přejde do podadresáře.
Opravy
  • SessionStart nespouští stav „running" — Claude Code posílá hook SessionStart při každém spuštění. Ten se dřív mapoval na running, takže každý nový tab okamžitě dostal oranžový puntík ještě před tím, než uživatel cokoliv napsal. Teď se SessionStart ignoruje.
  • Titulek tabu se po dokončení nezamění na „Claude Code" — Claude Code po skončení úkolu resetuje OSC titulek na ✳ Claude Code (idle stav). To přepisovalo smysluplný popis úkolu. Nově se blokuje pouze tento konkrétní idle titulek — ostatní OSC titulky (např. z /rename příkazu) projdou. Ověřeno přímým zachycením PTY výstupu: Claude Code emituje automaticky jen OSC 0;✳ Claude Code při startu a nic jiného.
  • Funguje /rename příkaz Claude Code — pokud v Claude Code napíšeš /rename "Můj úkol", titulek tabu se v Burrow aktualizuje. Předchozí příliš agresivní freeze blokoval i manuální přejmenování. Tip: /color pro změnu barvy.
  • Titulek tabu neukazuje název adresáře místo popisu úkolu — oprava v2 — předchozí oprava (pendingOscTitle jen při isAgentSession=true) zahazovala i startup OSC titulek Claude Code (✳ Claude Code), protože ten přichází PŘED prvním pollem (2 s). Nová logika: vždy bufferovat, ale smysluplný titulek (má mezeru nebo Unicode) nesmí být přepsán shell cwd šumem (krátké slovo bez mezer). Shell async precmd emituje OSC 2;work po trySend — to teď prohraje s ✳ Claude Code.
  • Přejmenování tabu (dvojklik) nefungovalo — input se zobrazil, ale nefocusoval — Tauri webview vyžaduje explicitní focus() před select(). Opraveno.
v2.3.0 červen 2026
Nové funkce
  • Fleet View — pohled na všechny workspace najednou v postranním panelu, přehled aktivních agentů přes celou flotilu.
  • Navigace notifikací na konkrétní tab — klik na notifikaci přepne přímo na tab, ze kterého přišla.
  • Widget využití tokenů (5h okno) — zobrazuje spotřebu Claude tokenů v posledních 5 hodinách přímo v záhlaví.
  • Deduplikace branch pickeru — výběr větve v postranním panelu nezobrazuje duplicitní záznamy.
  • Pravidlo delegování v CLAUDE.md — Burrow automaticky vloží instrukci burrow spawn do globálního ~/.claude/CLAUDE.md, aby agenti věděli jak delegovat práci.
Opravy
  • Claude chat — chyby oprávnění viditelné v UI — permission errory se zobrazují v chatu, ne jen v konzoli. Přidány klávesové zkratky Y/N pro rychlé potvrzení.
v2.2.0 červen 2026
Nové funkce
  • Podpora prohlížeče (browser tab) — nový typ tabu s embedded webview pro zobrazení localhost dev serverů nebo libovolné URL přímo v Burrow.
  • Drag & drop rozdělení pane — přetažením tabu na край existujícího pane ho rozdělíte horizontálně nebo vertikálně.
  • Oprava status hook sémantiky — sjednocení s modelem cmux: správné přechody mezi running/waiting/done bez falešných stavů.
v2.1.x červen 2026
Nové funkce
  • Claude chat jako plnohodnotný tab — chat s Claudem vedle terminálových tabů, s historií a persistencí přes restarty.
  • Vkládání obrázků přes ⌘V — vložení screenshotu do Claude chatu s náhledem před odesláním.
  • Fronta zpráv — můžete psát Claude další zprávu, zatímco ještě zpracovává předchozí. Zpráva se pošle automaticky po dokončení.
  • Widget využití tokenů — zobrazuje se hned jak existuje jakákoliv Claude session.
  • Centrum notifikací v záhlaví — zvoneček s přehledem posledních notifikací od agentů.
Opravy
  • Duplicitní done notifikace z idle pingers odstraněny.
  • Historie chatu se správně obnovuje po restartu, thinking bloky fungují.
  • Markdown render, /pr příkaz, tlačítko přerušení v Claude UI.
v2.0.x červen 2026
Nové funkce
  • Claude UI mode — nativní UI pro chat s Claudem, multi-session postranní panel, command suggestions, status line.
  • Permission approval UI — vizuální potvrzení oprávnění v Claude UI, dangerous mode toggle.
  • burrow set-progress — agenti mohou nastavit progress bar (0–100 %) pro dlouhé operace.
  • burrow set-status — custom textový status vedle status dotu v tabu.
  • Diff tab — záložka pro zobrazení git diffu přímo v Burrow.
  • tmux shim — burrow CLI simuluje základní tmux příkazy pro kompatibilitu se staršími skripty.
  • burrow log — agenti mohou logovat zprávy do Burrow bez výpisu do terminálu.
  • Session restore — daemon udržuje PTY session naživu přes restart Burrow.
  • Jump to unread — tlačítko pro přeskočení na první nepřečtenou notifikaci.
v1.0.0 červen 2026
První stabilní verze
  • Centralizovaný terminal status — sticky názvy tabů, jednotná logika stavů přes celou aplikaci.
  • Nové téma — tmavé téma s novými barvami, klikatelné ikony v postranním panelu.
  • Git panel vylepšení — lepší zobrazení diff, stage/unstage, commit.
  • Drag & drop reorder tabů — přetažení pro změnu pořadí bez artefaktů.
  • Worktree podpora — burrow worktree vytváří git worktree a otevírá ho jako nový workspace tab.
v0.x — beta 2026
Základ
  • Multi-workspace IDE s reálnými PTY (Rust portable-pty)
  • Split panes (horizontal / vertical), persistované tabu
  • Agent toolbar — barevná tlačítka pro spuštění agentů (Claude, Codex, Aider…)
